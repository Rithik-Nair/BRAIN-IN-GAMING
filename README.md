# 🧠 Brain in Gaming

An innovative project that enables users to play games using **brain signals**. This system analyzes brain activity such as focus, attention, and emotional states to control gameplay and provide detailed performance insights.

---

## 🚀 Features

- 🎮 Play games using brain commands
- 🧠 Real-time brain signal analysis
- 📊 Advanced analytics dashboard
- 📈 Visualizations:
  - Line Graphs (Emotional States)
  - Heatmaps
  - Bar Charts & Scatter Plots
  - Radar Charts (Focus, Stress, Calmness)
  - Dial Gauges (Engagement Levels)
- 🗂️ Game history tracking
- 🔐 User authentication (Login/Register)

---

## 🎯 Games Included

- **Mind Maze** → Card-based memory game  
- **Neuro Dash** → Endless runner (Dino-style)  
- **Cerebral Crossing** → Word search puzzle
- **Brain Beats** → Instrumental


---

## 🛠️ Tech Stack

- **Frontend:** Angular  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Other:** Brain signal processing (simulated / external integration ready) using BCI
